Project Ideas
1. Habit tracker
        An application where users can create daily habits, track their progress, and view statistics.
   Features
        Registration and Login
        Create Habits
        Mark Habits as Complete
        Weekly Statistics
        Progress Calendar
        Reminders
    CRUD
        Habits
        Categories
        Goals
2. Inventory system for a store
        An application designed to assist with inventory management, customer tracking, and sales, while also providing oversight of missing stock or items running low.
    Features
        Inventory Management
        Sales Tracking
        Customer Management
        Low Stock Alerts
    CRUD 
        Products
        Categories
        Sales
        Customers
3. Personal finance app
        Income and Expense Tracking
    Features
        Record Expenses
        Categories
        Financial Charts
        Monthly Budget
        Export Reports
    CRUD
        Transactions
        Categories
        Budgets
4. Musical Progress Tracking Platform (Guitar Study)
        A tool for music teachers or independent students to organize their practice routines, track the songs they are learning, and measure their progress.
    Features
        Practice session logging
        Teacher profiles
        Assignments
        Progress tracking and statistics
        Categories
        Student Profile
    CRUD
        Sessions
        Records
        Monthly Statistics
5. Table Reservation and Ordering System for Restaurants
        A responsive (mobile-first) web application enabling restaurant diners to view the menu, book a table, and place an order in advance.
    Features
        Restaurant Staff Profiles
        Table Reservations
        Daily Reservation List
        Name Registration
        Customer Database
    CRUD
        Records
        Reservations
        Categories
        Reservation Types
